## Development Setup

### Prerequisites

- Install [node.js] which you need to use [npm]

### Setting Up a Project and running (clientside)
Clone git repo
```
git clone https://github.com/Koksiarz100/CineCritique
```
Go to client
```
cd client
```
Run next.js run
```
npm run dev
```

[node.js]: https://nodejs.org/en
[npm]: https://www.npmjs.com
